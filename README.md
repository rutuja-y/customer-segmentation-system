# Customer Segmentation System

## Overview
Customer Segmentation System is a machine learning project that segments customers based on demographics, purchasing behavior, and campaign responses. It helps businesses understand customer groups and create targeted marketing strategies.

## Problem Statement

Businesses often spend large amounts on marketing campaigns without understanding which customers are most likely to respond.

Customer Personality Analysis solves this problem by:

- Identifying customer segments based on purchasing behavior
- Understanding customer preferences and spending patterns
- Helping businesses personalize product recommendations
- Improving campaign targeting
- Increasing marketing ROI

## Dataset Features
The dataset includes:

- Customer demographics (Age, Education, Marital Status, Income)
- Purchase behavior
- Product spending
- Website visits
- Campaign responses
- Customer recency and complaints

## Technologies Used

- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Scikit-learn**
- **Kneed**

## Workflow
1. Data Preprocessing  
2. Exploratory Data Analysis  
3. Feature Engineering  
4. Customer Segmentation using K-Means  
5. Cluster Analysis and Business Insights  

## Key Outcome
Identified 4 customer segments:
- Low-value inactive customers
- Loyal premium customers
- Browsers / hesitant buyers
- High-spending young buyers

## Business Impact
This analysis helps businesses:
- Improve customer targeting
- Personalize campaigns
- Increase customer retention
- Optimize marketing efforts
